# 智能合盘与定盘系统完整开发计划 (V2.2)

本计划整合了双通道定盘（三角定位+MBTI）需求，并包含前后端详细技术文档的编写规划。

## 1. 核心目标
构建集“智能定盘”与“深度合盘”于一体的关系咨询系统，解决“时辰未知”痛点，提供非宿命论的 AI 关系指导。

## 3. 功能模块详解

### 模块 A：智能定盘向导 (前端重点)
*   **双通道入口**：用户自选“三角定位法”（默认高亮）或“MBTI 定盘”。
*   **三角定位法**：基于 `rectification_weights.json` 配置，计算 12 时辰得分。
*   **MBTI 映射**：基于 `mbti_mapping.json` 筛选候选时辰。

### 模块 B：深度合盘分析 (前后端协作)
*   **数据预处理**：前端计算基础参数，后端/AI 处理飞星互涉。
*   **AI 交互**：后端提供流式接口，前端渲染 Markdown + 图表。

### 模块 C：关系档案管理 (后端重点)
*   **存储**：PostgreSQL 存储完整的定盘推导路径和置信度，便于后续优化算法。
